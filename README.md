# 🎓 Smart Student Life System

✨ *A simple yet practical AI-driven system built to solve real student problems.*

---

## 🚀 Overview

Managing studies, productivity, and finances together can be overwhelming for students. I personally faced this issue — switching between different apps and still not feeling organized.

So, I built the **Smart Student Life System** — a single platform that helps manage all three areas in a simple and effective way using data-driven logic.

---

## 🌟 Key Features

### 📚 Smart Study Planner

* Select subjects based on your needs
* Automatically generates a study plan
* Allocates time based on difficulty and priority
* Creates a structured daily timetable
* Visualizes time distribution using charts

---

### 📊 Productivity Tracker

* Add daily study hours, sleep, and focus score
* Track your performance over time
* Visualize productivity trends with graphs
* Predict future performance using simple logic
* Get insights based on your habits

---

### 💰 Finance Planner

* Record income and expenses
* Categorize spending (food, shopping, transport, etc.)
* Calculate total savings instantly
* Visualize expenses using pie charts
* Identify overspending patterns

---

## 🧠 How It Works

### 📌 Study Planning Logic

Time is distributed using:
**Difficulty × Priority = Weight**
Higher weight → More study time

---

### 📌 Productivity Prediction

Uses a simple trend line to estimate future performance.

---

### 📌 Finance Analysis

**Savings = Income − Expenses**
